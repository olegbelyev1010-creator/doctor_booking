# ДокторОнлайн — ИИ-агент записи к врачу

Прототип веб-сервиса: чат с агентом на **OpenRouter**, который записывает на онлайн-консультацию к врачу из базы клиники.

## Возможности

- Веб-чат в стиле регистратуры клиники
- Подбор врача по **27 специальностям**
- Поиск с учётом **даты и времени** (только врачи со свободными слотами в окне)
- До **3 врачей** в ответе: ближайший слот + лучший рейтинг
- Запись на консультацию с сохранением в `data/bookings.json`

## Стек

- Node.js 18+ (встроенный HTTP-сервер, без внешних зависимостей)
- OpenRouter API (`openai/gpt-5.4` по умолчанию)
- HTML / CSS / JavaScript (чат)

## Быстрый старт

```bash
git clone https://github.com/olegbelyev1010-creator/doctor_booking.git
cd doctor_booking

cp .env.example .env
# Укажите OPENROUTER_API_KEY в .env
```

Откройте http://localhost:3000

## Переменные окружения

| Переменная | Описание |
|------------|----------|
| `OPENROUTER_API_KEY` | Ключ API с [openrouter.ai/keys](https://openrouter.ai/keys) |
| `OPENROUTER_MODEL` | Модель (по умолчанию `openai/gpt-5.4`) |
| `PORT` | Порт сервера (по умолчанию `3000`) |

## База врачей

Файл `data/doctors.json` генерируется скриптом:

```bash
npm run generate:doctors
```

- **27 специальностей** (аллерголог-иммунолог, терапевт, стоматолог-терапевт и др.)
- **2–10 врачей** на специальность (случайно при генерации)
- Слоты **до конца июня** (с завтрашнего дня)
- Добавить слоты к существующей базе: `npm run extend:slots`

Поля врача: специальность, пол, стаж, рейтинг, отзывы, цена, свободные слоты.

## Как работает агент

Агент использует **function calling**:

| Инструмент | Назначение |
|------------|------------|
| `search_doctors` | Поиск по специальности, времени, рейтингу, стажу |
| `get_doctor_slots` | Свободные слоты выбранного врача |
| `book_appointment` | Создание записи |
| `list_my_bookings` | Записи пациента по телефону |
| `get_specialties` | Список специальностей |

Поведение: как регистратура — без расспросов о симптомах; пол врача не обязателен; при указании времени показываются только врачи со свободными слотами.

## Пример диалога

> **Пациент:** Запишите к терапевту в субботу с 11 до 14  
> **Регистратура:** Подобраны варианты на ближайшее время с лучшим рейтингом — 3 терапевта со свободными слотами в субботу 11:00–14:00  
> **Пациент:** К Ивановой на 12:00, Петров Иван, +79001234567  
> **Регистратура:** Запись подтверждена

## API

| Метод | Путь | Описание |
|-------|------|----------|
| `GET` | `/api/health` | Проверка сервера |
| `GET` | `/api/specialties` | Список специальностей |
| `GET` | `/api/doctors` | Поиск врачей (query-параметры) |
| `POST` | `/api/chat` | Сообщение агенту |
| `POST` | `/api/chat/reset` | Сброс сессии |
